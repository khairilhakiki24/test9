AsetDesa Pro V4

Buku Panduan Penggunaan & Modifikasi (Standar ASN)

1. Fitur Utama & Standar Audit

Sistem Mutasi (Anti-Hapus Permanen)

Sesuai Permendagri tentang pengelolaan aset, data barang tidak boleh dihapus secara permanen. Aplikasi ini menggunakan sistem Soft-Delete. Aset yang rusak atau dilelang akan dipindahkan ke Buku Mutasi lengkap dengan tanggal dan alasannya.

Scanner QR Code Terintegrasi

Aplikasi dilengkapi generator Label QR yang dapat dicetak massal. Untuk memantau barang fisik, cukup tekan tombol Scan QR Kamera, arahkan kamera HP/Tablet, dan spesifikasi aset akan muncul otomatis di layar.

2. Manajemen Data & Keamanan (Sangat Penting)

Backup & Restore Otomatis via Excel

Aplikasi berjalan 100% secara offline di perangkat Anda. Agar data aman jika perangkat rusak, lakukan hal berikut:

  1. Buka menu Pengaturan.
  2. Klik Ekspor Data untuk mengunduh file berekstensi .xlsx. File ini berisi seluruh database (Buku Aset, Mutasi, Kategori, dan Pengaturan Tema).
  3. Simpan file tersebut di Flashdisk atau Google Drive Anda.
  4. Jika Anda berpindah laptop, cukup masuk ke Pengaturan lagi dan klik Impor Data, lalu pilih file Excel tadi. Seluruh aplikasi akan pulih seperti semula.

3. Modifikasi Tema & Identitas (Tanpa Coding)

Anda tidak perlu mengerti bahasa pemrograman untuk mengubah tampilan dasar aplikasi ini. Akses menu Pengaturan di dalam aplikasi untuk:

  • Mengganti Nama Instansi.
  • Menyematkan Logo (bisa berupa Link URL gambar atau Upload File dari komputer).
  • Mengganti Warna Tema Utama menggunakan Color Picker. Seluruh tombol, garis, dan highlight aplikasi akan mengikuti warna yang Anda pilih seketika.

4. Panduan Modifikasi Lanjutan (Untuk Pembeli CodeCanyon)

File utama aplikasi ini digabung dalam satu file index.html untuk mempermudah preview. Namun, kami telah menyematkan komentar pemisah standar CodeCanyon untuk mempermudah Anda memecah file tersebut ke struktur folder assets.

Langkah 1: Memisahkan CSS

Buka file index.html, cari komentar HTML yang berbunyi <!-- FILE: source/assets/css/style.css START -->. Potong seluruh kode <style>...</style> di area tersebut dan tempelkan ke dalam file baru bernama style.css di folder assets/css. Ganti tag di html menjadi <link rel="stylesheet" href="assets/css/style.css">.

Langkah 2: Memisahkan JavaScript

Gulir ke bagian paling bawah index.html. Temukan komentar // FILE: source/assets/js/app.js START. Potong kode di dalam tag <script>...</script>, buat file app.js di folder assets/js, dan panggil menggunakan tag script eksternal.

5. Panduan Instalasi (Hosting CPanel)

  1. Siapkan domain/subdomain yang sudah memiliki Sertifikat SSL aktif (wajib HTTPS agar fitur Kamera dan PWA berfungsi).
  2. Login ke cPanel Hosting Anda, masuk ke menu File Manager.
  3. Buka direktori public_html atau folder subdomain Anda.
  4. Upload seluruh file (index.html, manifest.json, sw.js, dan folder assets jika Anda menambahkan icon).
  5. Selesai! Aplikasi sudah dapat diakses dan diinstal (Add to Home Screen) melalui HP atau PC mana pun.
